# Demo

## Basic Usage

```bash
# 1. initialize venv by windows powershell 
python.exe -m venv .venv 
.\.venv\Scripts\activate 

# 2. install package dependencies
python.exe -m pip install -r requirements.txt
python.exe -m pip install --upgrade pip

# 3. run script
python main.py

# 999. close env
deactivate
```
